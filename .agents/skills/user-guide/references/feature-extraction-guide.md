# 機能抽出ガイド & Turtle Video 機能カテゴリ

## 機能抽出の対象

| 対象 | 参照先 | 抽出する情報 |
|------|--------|-------------|
| UI機能 | `src/components/` | ボタン、スライダー、モーダルなどの操作可能な要素 |
| セクション機能 | `src/components/sections/` | 各セクション（クリップ、BGM、ナレーション、キャプション、プレビュー）の機能 |
| モーダル | `src/components/modals/` | 各モーダル（AI、設定、保存・読込、キャプション設定）の機能 |
| ストア | `src/stores/` | 状態管理で提供される機能（保存、ロック、ミュートなど） |
| フック | `src/hooks/` | カスタムフックが提供する機能（再生、エクスポート、AI等） |
| 定数 | `src/constants/index.ts` | 選択肢、上限値などユーザーに関わる定数 |
| 型定義 | `src/types/index.ts` | 設定可能なパラメータの範囲を確認 |
| 仕様書 | `spec.md` | 機能一覧と仕様 |

## 抽出方法

1. `spec.md` の機能一覧を基本リストとする
2. `src/components/` 配下のコンポーネントから UI 操作を抽出
3. `src/stores/` の各アクションでユーザー操作に対応するものを確認
4. 上記を統合して、機能一覧を完成させる

## Turtle Video の機能カテゴリ

ユーザーガイドのセクション構成は以下を推奨する：

1. **クリップ（動画・画像）の管理**
   - アップロード、並べ替え、削除
   - トリミング
   - ボリューム調整、ミュート
   - フェードイン/アウト
   - スケール（拡大/縮小）と位置調整
   - ロック機能

2. **BGM（バックグラウンドミュージック）**
   - 音楽ファイルの追加
   - 開始位置と開始タイミングの設定
   - ボリューム調整とフェード

3. **ナレーション**
   - 音声ファイルの追加
   - 開始位置と開始タイミングの設定
   - ボリューム調整とフェード

4. **AIナレーション**
   - APIキーの設定方法
   - スクリプトの自動生成
   - 音声合成とボイス選択
   - 生成した音声の保存

5. **キャプション（字幕）**
   - 字幕の追加・編集
   - スタイル設定（サイズ、位置）
   - タイムライン上の配置

6. **プレビューと再生**
   - 再生/一時停止/停止
   - シークバーの操作
   - リソースリロード

7. **動画のエクスポート**
   - 動画ファイルの作成
   - ダウンロード

8. **プロジェクトの保存・読み込み**
   - 自動保存の設定
   - 手動保存とスロット管理
   - プロジェクトの読み込み・削除

9. **設定**
   - APIキー管理
   - ログ閲覧
